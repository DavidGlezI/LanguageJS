OVNI: Lenguaje tipo DSL

Lenguaje tipo DSL inspirado en LOGO. Cuenta con analizador léxico usando regex y un parser para analizar la sintaxis implementado con descenso recursivo. Se recomienda abrir la consola para ver mensajes del parser.

Creado por: David Alejandro González Ibarra A01633817

Instrucciones de uso del lenguaje OVNI:

Reconoce los siguientes comandos para pintar la pantalla:

    • Avanzar (1-50)

      Permite mover el ovni hacia arriba. Importante -> ()

      Retroceder (1-50)

      Permite mover el ovni hacia abajo. Importante -> ()

      Vuelta derecha (1-50)

      Permite mover a la derecha en angulo de 90 grados. Importante -> ()

      Vuelta izquierda (1-50)

      Permite mover a la izquierda en angulo de 90 grados. Importante -> ()

      Pintura amarillo/rojo/verde/azul

      Permite cambiar la pintura de toda la linea que dibuja para el siguiente comando

      Centrar Ovni

      Permite centrar el ovni y limpiar los dibujos.

      Para repetir un comando se ponde dentro del paréntesis
      Ejemplo: Avanzar (1-50, repetir #)
      Acepta comandos seguidos uno de otro sin importar espacios o saltos de linea
